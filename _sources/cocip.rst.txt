CoCiP Data
==========

CoCiP data is processed as datasets of contrail segments. The analysis has been
performed on output from the ``pycontrails`` package [1]_.

.. [1] Shapiro, M., Engberg, Z., Teoh, R., Stettler, M., Dean, T., 2023. pycontrails: Python library for modeling aviation climate impacts. https://doi.org/10.5281/zenodo.10182539
